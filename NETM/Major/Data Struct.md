3. 算法效率
    1. 	时间复杂度`T(n)=O(f(n))`

		O(1), O(n), O($n^2$), O($\log{2}$⁡), O($2^n$)  
        分别为
        常量阶，线性阶，平方阶，对数阶，指数阶
		
		- 时间复杂度O(1)
		
        `x++;`
		
		- 时间复杂度O(n)
        
        ``` c
        for (i = 1; i < n; i++){
            x++;
        }
        ```
		
		- 时间复杂度O(n^2 )
        
        ```c
        for (i = 1; i < n; i++){
            for (j = 1; j < n; j++){
                x++;
            }
        }
        ```
