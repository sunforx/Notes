## 命令语法惯例

命令语法遵循的约定与IOS命令手册使用的惯例相同。命令手册对这些惯例的描述如下。
- 粗体字表示照**原样输入**的命令和关键字，在实际的设置和输出（非常规命令语法）中，粗体字表示命令由用户手动输入（如show命令）。
- 斜体字表示**用户应提供**的具体值参数。
- 竖线（|）用于分隔**可选的**、互斥的选项。
- 方括号（[]）表示**任选项**。
- 花括号（{}）表示**必选项**；
- 方括号中的花括号（[{}]）表示必须在任选项中选择一个。
